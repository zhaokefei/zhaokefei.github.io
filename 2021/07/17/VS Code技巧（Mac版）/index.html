VS Code常用功能（Mac版）

发表于 2021-07-17 阅读次数： Valine：

快速打开文件

快捷键：⌘P

切换最近使用的文件夹

快捷键：⌃R

命令行内快速使用vscode打开文件夹

注，若未安装命令行命令，可进入命令面板（Cmd+Shift+P）后选择

打开内置终端

快捷键：⌃`

打开关闭侧边栏

快捷键：⌘B

竖向分屏

快捷键：⌘\

分屏间快速切换

快捷键：⌘1、⌘2、⌘3

向上/向下复制

快捷键：⇧⌥↑或⇧⌥↓

向上/向下移动

快捷键：⌥↑或⌥↓

转到文件中的符号

快捷键：⇧⌘O

使用添加冒号，对种类进行分类

代码格式化

快捷键：⇧⌥F

代码折叠

快捷键：⌥⌘[和⌥⌘]
